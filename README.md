## Development

Run

```sh
$ npm install

$ npm run dev
```

Visit `localhost:5000`

Or with Docker Compose

```sh
$ cp docker-compose.override.yml.dist docker-compose.override.yml

$ docker-compose up
```

Visit `localhost:7000`
